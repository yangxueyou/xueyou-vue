# 3. render: h => h(App)

**使用 App 作为这个 Vue 实例的 template（同时一并了注册 App 组件**

```javascript
render: function (createElement) {
    return createElement(App);
}


render (createElement) {
    return createElement(App);
}

render (h){
    return h(App);
}

render: h => h(App);
```
